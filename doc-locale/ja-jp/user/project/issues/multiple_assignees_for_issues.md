所有権を共有する大規模なチームでは、誰がイシューに取り組んでいるか、誰がすでに完了しているか、または誰がまだ開始していないかを追跡することが難しい場合があります。

複数の[assignees](managing_issues.md#assignees)をイシューに追加して、追跡しやすくし、誰が責任を負うかをより明確にすることができます。

イシューに対する複数のアサイン先により、コラボレーションが円滑になり、共有責任を明確に表示できます。すべてのアサイン先がチームのワークフロー全体に表示され、（単一のアサイン先の場合と同様に）通知を受信するため、コミュニケーションと所有権が簡素化されます。

アサイン先が自分の作業を完了すると、自分自身をアサイン先から削除し、自分のタスクが完了したことを明確にします。
